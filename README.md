# Hexfield

Inspired by drag-and-drop coding platforms like Scratch, Hexfield is a drag-and-drop IDE for creating HTML documents.

To add an element, drag it from the column on the left into the coding area on the right. To delete an element, drag it from the coding area to the column on the left.

Many element types are represented in the left column. For types that aren't listed, two custom element types are available. One custom type allows for child elements; the other does not.

Elements may be given attributes, which are boxes with rounded tops. There is one special attribute, **style**, with its own unique block type: bexagons.

Buttons at the top-right allow you to render the current HTML file, export it, or temporarily lock the blocks in place.

You can create or upload multiple files and switch between them. HTML pages listed in the editor may link to each other.
